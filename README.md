# Kino

 *Film in Terminal*

A simple CLI tool to search, stream and track films/TV shows.

![Demo](assets/example.gif)

## Dependencies

- **mpv** - Media player
- **Go 1.25+** (for building from source)

## Installation

```bash
git clone https://github.com/astyrisk/kino.git
cd kino
go build -o kino .
```

## Usage


```bash
./kino
```
or 
```bash
./kino "The Matrix"
```
