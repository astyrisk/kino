# Kino

 *Film in terminal*

A simple CLI tool to search and stream movies and TV shows using mpv player.

![Demo](assets/example.gif)

## Dependencies

- **mpv** - Media player
- **Go 1.25+** (for building from source)

## Installation

```bash
git clone https://github.com/astyrisk/kino.git
cd kino
go build -o kino .
```

## Usage

### Interactive Mode

```bash
./kino
```

### Search from Command Line

```bash
./kino "The Matrix"
```
